# Quality That Compounds
### A Guide to SDLC Automation in the AI & Agentic Era
*Ankit Sobti, CTO & Co-Founder, Postman*

A long-form interactive web document built for Postman, covering the full SDLC Automation framework across five chapters.

## Structure

```
index.html   # Single self-contained file — all styles, scripts, and images inline
```

## Features

- Sticky side navigation with active section tracking
- Floating chapter bar that updates on scroll
- Tabbed interface for the three-phase 90-day plan (Chapter 5)
- 17 inline images (base64-encoded, no external dependencies)
- Fully self-contained — single HTML file, no build step required

## Usage

Open `index.html` directly in a browser, or serve from any static host (GitHub Pages, Netlify, etc.).

### GitHub Pages

1. Push to a repository
2. Go to Settings → Pages
3. Set source to `main` branch, `/ (root)`
4. Site will be live at `https://<username>.github.io/<repo>/`
